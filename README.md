# journal
Journal is a simple fullstack web application that allows users to sign up, log in, view their profile, and record personal thoughts. Built with Vue.js on the frontend, Django on the backend, and SQLite as the database, this project is designed for practicing and strengthening fullstack web development skills.

🔧 Tech Stack
Frontend: Vue.js

Backend: Django (REST API)

Database: SQLite

✨ Features
User Signup and Login

JWT-based Authentication

Profile Viewing

Create, View, and Manage Personal Thought Entries

🚀 Purpose
This project is a personal exercise to deepen my understanding of fullstack development using modern technologies. It focuses on handling authentication, API integration, and component-based frontend architecture.
